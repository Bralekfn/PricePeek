Privacy Policy

PricePeek does not collect or store personal user data.

We fetch data from CoinGecko to display cryptocurrency prices and gas fees. All data is retrieved from public APIs and no user information is stored or shared.

The bot may temporarily use Discord identifiers (such as user IDs or server IDs) only to provide the requested functionality during the session. None of this data is stored persistently or used for tracking.

We only store the channel IDs of voice channels created by the bot in order to update their names with live price data. This is necessary for the bot to function properly and provide accurate information in real time.

If Discord API behavior changes in the future, this policy will be updated accordingly.
